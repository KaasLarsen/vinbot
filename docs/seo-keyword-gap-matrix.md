# SEO keyword-gap matrix (DK vin × Vinbot)

Kort mapping fra typiske danske søgeclusters til **primær guide** på Vinbot og **note** efter gap-arbejde (maj 2026). Clusters kan komme fra keyword- eller research-eksporter (fx Gemini): her bruges de som **checkliste**, ikke som krav om én ny URL pr. enkeltfrase.

| Cluster / søgeintent | Primær URL (slug) | Note |
|---------------------|-------------------|------|
| Vin på tilbud, rødvin tilbud | `koeb-vin-online-sadan-holder-du-styr-paa-det` | Udvidet med dedikeret afsnit + FAQ; brug Vinbot-søg til aktuelle priser |
| Papvin, bedste papvin, BIB | `bedste-box-vin` | Synonymer + FAQ; «papvin» i titel/meta/brødtekst |
| Værtsgave, hvad koster en god flaske vin | `bedste-vin-til-gave`, `gavevin-sadan-vaelger-du-den-rigtige-flaske` | Eksplicit DK-prisbånd i kr |
| Vivino app, ratings i supermarked | `vivino-app-til-vin-anmeldelser` | Ny neutral guide; link fra vinbegreber |
| Grænsehandel vin (generelt, ikke ét brand) | `vin-graensehandel-for-danskere` | Ny neutral informationsguide |
| Hvor længe holder åbnet vin (samlet) | `hvor-laenge-holder-aabnet-vin` | Pillar med links til rød/hvid/boble/boks |
| Åbnet rød/hvid/boble/boks/rosé (detalje) | `hvor-laenge-holder-rodvin`, `hvor-laenge-holder-hvidvin`, `hvor-laenge-holder-bobler-og-champagne`, `hvor-laenge-holder-boks-vin`, `hvor-laenge-holder-rosevin` | Uændret kerne; fælles krydslink til pillar i sidefødder |
| Vin i karaffel / dekantering tid | `hvor-laenge-holder-vin-i-karaffel`, `sadan-dekanterer-du-vin` | Krydslink til åbnet-vin-pillar og holdbarhed pr. type |
| Genstande i flaske / glas | `hvor-mange-enheder-alkohol-i-et-glas-vin`, `hvor-mange-glas-i-en-flaske-vin` | Allerede stærkt |
| Serveringstemperatur | `opbevaring-af-vin-temperatur-og-aabnet-flaske`, `sadan-serverer-du-vin` | Allerede stærkt |
| Italiensk rødvin, Amarone, Barolo | `bedste-italiensk-rodvin`, `vinregion-italien`, `nebbiolo-fra-barolo`, `bedste-rodvin-under-300-kr` | Allerede stærkt |
| Dansk vin, vingård | `bedste-dansk-vin` | Allerede stærkt |
| Vinpakker / smagekasser | `bedste-vin-smagekasser`, `koeb-vin-online-sadan-holder-du-styr-paa-det` | Allerede stærkt |
| Kalorier, sulfit, hovedpine | `hvor-mange-kalorier-i-vin`, `hvad-er-sulfit-i-vin`, `hvad-er-tanniner` | Delvist; ingen obligatorisk ny URL |
| Madlavning vin, risotto, tapas | `sadan-bruger-du-vin-til-sauce-og-simren`, `vin-til-risotto`, `vin-til-tapas` | Allerede stærkt |
| Fest / bryllup mængde | `hvor-meget-vin-til-fest`, `hvor-meget-vin-til-bryllup` | Allerede stærkt |

**Strategi:** affiliate-side prioriterer *commercial investigation* og dybe guides; transaktionelle head terms («billig rødvin») overlades til store webshops — Vinbot fanger trafik tidligt og sender videre via sammenligning.
