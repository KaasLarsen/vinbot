## Fransk vinviden møder soja og kimchi

Klassisk **fransk og italiensk** parring hjælper kun delvist, når menuen er **takeaway**: dumplings, sushi, ramen, pho og **rød karry**. Udfordringen er **soja** (umami + salt), **ingefær**, **chili**, **sesam** og **fermenterede** elementer som **kimchi** og **miso**.

**Tung rødvin med høj tannin** møder ofte soja og fiskesauce og kan smage **metallisk og bittert**. **Let hvid med syre**, **off-dry riesling**, **grüner veltliner**, **gewürztraminer** og **tør bobler** er ofte de reelle redningsmænd — plus **afkølet gamay** til mildere retter.

## Hurtig parringstabel

| Ret / takeaway | Bedste vinspor | Undgå |
|----------------|----------------|-------|
| **Sushi & sashimi** | Tør **riesling**, **albariño**, **cava brut**, **kølig pinot** | Ung stram cabernet, tung egetræs-chardonnay |
| **Dumplings** (svine, grønt) | **Riesling** (halvtør ved sød soja), **grüner**, **prosecco extra dry** | Kraftig ung malbec |
| **Ramen** (miso, soja, fed bouillon) | **Off-dry riesling**, **gewürztraminer**, let **orangevin** | Tanninrig rød uden mad |
| **Rød thai-/karry** | **Riesling halbtrocken**, **gewürz**, **demi-sec bobler** | Tør syrah, ung bordeaux |
| **Koreansk (kimchi, BBQ)** | **Off-dry hvid**, **rosé**, **afkølet gamay** | Tung egetræs-rød |
| **Wok / stir-fry** | **Grüner**, **riesling**, **cinsault-rosé** | 14,5 % ung shiraz |
| **Friture (tempura, nuggets)** | **Cava**, **crémant**, **prosecco** | Tung rød |

## Hvorfor riesling og grüner vinder

- **Syre** skærer **fedt** og **salt**.
- **Lille restsødme** (halvtør, off-dry) **tæmmer chili** bedre end knastør hvid.
- **Grüner veltliner** tåler **ingefær og grøntsager** — se [vinregion Central- og Østeuropa](/guides/vinregion-europa-central-og-oest).
- **Gewürztraminer** matcher **aromatisk** mad (karry, citrongræs).

## Dumplings — gyoza, bao og wonton

**Sød soja-dip** → **extra dry prosecco** eller **halvtør riesling**. **Kraftig hvidløg og chili** → mere **sødme i vinen** eller **bobler**. **Stegte dumplings med fedt** → **cava** eller **crémant** (samme logik som [bobler til takeaway](/guides/bobler-til-takeaway-og-fastfood)).

## Ramen og pho

**Fed bouillon** kræver **syre og evt. sødme** — ikke tannin. **Pho** med urter: **riesling** eller **albariño**. **Miso-ramen**: **gewürz** eller **off-dry chenin**.

## Rød karry og stærk wok

Jo **stærkere chili**, jo mere **restsødme eller bobler** — se [vin til krydret og stærk mad](/guides/vin-til-krydret-og-staerk-mad). **Afkølet gamay** kan fungere til **mild** koreansk eller **ikke-for-stærk** wok.
