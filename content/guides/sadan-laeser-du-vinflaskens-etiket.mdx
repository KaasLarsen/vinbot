## Hvorfor etiketten stadig betyder noget

På nettet får du ofte **smagsnoter og anmeldelser**; i **fysiske butikker** er etiket og bagside det hurtige filter. Denne side går **dybere** end oversigten i [vinbegreber](/guides/vin-begreber-i-praksis) — men du kan med fordel læse begge.

## 1. Område først (appellation)

**Navnet på et sted** (fx *Chianti*, *Rioja*, *Bourgogne village*) fortæller ofte mere end **druen**, fordi **reglerne** for området styrer **udbytte**, **druer** og **lagring**. EU bruger **PGI/PDO** (beskyttede betegnelser) — kig efter **oprindelsesbetegnelse** på forsiden eller bagsiden.

## 2. Når druen står på etiketten

I mange **nye verdensvine** og i **Alsace** står **druen** tydeligt. I **klassisk Europa** kan druen **mangle** fordi appellationen **definerer** den. Tvivl? Søg appellationen kort på telefonen eller brug [Vinbot-søgningen](/) på navnet.

## 3. Årgang: ung vs moden

**Årgang** er **vejret det år** druerne blev høstet. I **kølige eller ustabile** områder varierer kvaliteten mere. **Unge røde** kan være **mere tannin**; **ældre** kan være **mere sammensatte** — men **holdbarhed** afhænger af **vin** og **opbevaring**. Se [opbevaring og temperatur](/guides/opbevaring-af-vin-temperatur-og-aabnet-flaske).

## 4. Alkoholprocent som pejlemærke

**Højere alkohol** (typisk **14 %+**) føles ofte **varmere** og **fyldigere**; **lavere** kan føles **lettere**. Det siger **ikke** alt om kvalitet — men hjælper til **mad** (fx **krydderi** og **fedme** tåler ofte mere krop).

## 5. Tør, halvtør, brut, extra dry …

På **bobler** kan **extra dry** være **lidt sødere** end **brut** — læs [bobler-guiden](/guides/bobler-champagne-cava-prosecco-og-cremant). På **hvide** betyder **trocken/feinherb** (tysk) eller **sec/demi-sec** (fransk) noget for **restsødme** — se [vinbegreber](/guides/vin-begreber-i-praksis).

## 6. Reserve, Grand Cru, Old Vines …

- **Reserva/riserva/crianza** = ofte **regelstyret** lagring i **landet** — brugbart **hint**, ikke **garanti**.  
- **Grand cru**, **premier cru** osv. = **franske hierarkier** med **specifikke** betydninger pr. område.  
- **Old vines / vieilles vignes** = ofte **marketing** med **varierende** definition — smag tæller.

## 7. Økologi og symboler

**Økologisk** og **biodynamisk** handler om **markpraksis** og **certificering** — ikke automatisk **smagsstil**. [Naturvin](/guides/naturvin-hvad-er-det) er et **bredere** begreb end øko.

## Kort checkliste i butikken

1. **Område / appellation** — kontekst og stil.  
2. **Årgang** — især i ustabile zoner.  
3. **Alkohol** — krop og varme.  
4. **Sødme/tørhed** — især tysk riesling og bobler.  
5. **Producent** — gentagelse giver erfaring.

Brug derefter forsiden til at **sammenligne pris** på konkrete flasker.
