Sangria med rødvin er spansk sommerklassiker, hvor **rødvin er selve retten** — frugt, sukker og sodavand bygger ovenpå flasken. Det er ikke madlavning i gryden, men en opskrift hvor vin er uundværlig base — som [rødgrød med portvin](/opskrifter/roedgroed-med-portvin) er dessert med vin som hovedingrediens. Ung tempranillo eller gamay fungerer bedst — undgå tung, fadlagret rød der bliver bitter i bowlen.

## Rødvin som base — ikke tilbehør

Vælg frugtig, ung rød du ikke er bange for at fortynde — det er pointen med sangria. Tempranillo, garnacha eller gamay giver bær og blød struktur. Lad frugt og vin trække sammen minimum 2 timer i køleskab; natten over giver dybere smag. Sodavand tilføjes sidst ved servering — ellers flad smag og døde bobler. Se [vin i cocktails](/guides/vin-i-cocktails-spritz-og-drikke).

## Frugt, balance og servering

**Frugt:** Appelsin og æble er klassisk — tilføj sommerbær eller fersken i sæsonen.

**Spiritus:** Valgfri brandy giver dybde — pas på ikke at overskygge vinen.

**Is:** Server kold i store glas med frugt og mynte — sangria skal forfriske.

**Balance:** Smag til — for sød, tilsæt citron; for tør, tilsæt honning eller sukker.

Til tapas: [chorizo i rødvin](/opskrifter/chorizo-i-rodvin) og [marinerede oliven med hvidvin](/opskrifter/marinerede-oliven-med-hvidvin). Se [vin til spansk mad](/guides/vin-til-spansk-mad).

## Fejl og opbevaring

Undgå tung cabernet — bitter og tannin i bowle. Ingen trækketid giver rå alkoholsmag. Sodavand for tidligt ødelægger friskhed. Bowlen holder 1–2 dage i køleskab uden sodavann — tilsæt bobler ved servering. Lav dagen før festen; tilsæt sodavann når gæster ankommer.

## Vin i glasset

Brug samme type rød som i bowlen — eller server en frisk hvid sangria-variant om sommeren. Se [vin til sommer](/guides/vin-til-sommer).
## Sommerfest og variationer

Sangria er fleksibel — tilpas frugt efter sæson og smag til sødme inden sodavand tilsættes. Hvid sangria med hvidvin og fersken er et sommeralternativ; denne røde version passer til tapas og grill. Lav dobbelt batch til havefest — bowlen er billigere end cocktails og alle kan tage selv. Opbevar uden sodavann og is; tilsæt begge dele ved ankomst så drinken forbliver frisk. Til børn og bilister: lav alkoholfri version med rød saft og samme frugt — samme feststemning uden vin.
## Festbowle og servering

Sangria skal være kold og frisk — is tilføjes ved servering, ikke timer i forvejen. Frugt skal trække minimum to timer; ellers smager bowlen af rå vin og sukker. Brug stor kande eller bowl så gæster kan se frugten. Til børn og bilister: lav parallel alkoholfri version med rød saft og samme frugt. Sangria er spansk sommertradition — server med tapas som [marinerede oliven med hvidvin](/opskrifter/marinerede-oliven-med-hvidvin) og brød.
