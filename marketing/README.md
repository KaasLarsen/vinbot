# Marketing-materiale (ikke på hjemmesiden)

## Video til sociale medier

**Fil:** `vinbot-saadan-bruger-du.mp4`  
**Format:** MP4, 1280×720, ~42 sek., klar til Facebook/Instagram.

Genbyg efter nye skærmbilleder:

```bash
node scripts/build-vinbot-walkthrough-video.mjs
```

### Forslag til Facebook-tekst

> Vinbot hjælper dig med at finde vin til maden — og sammenligne priser hos danske forhandlere.  
> 🔍 Søg på ret, drue eller budget  
> 📖 Læs guides og opskrifter med vin  
> 🧊 Find vinkøleskab når samlingen vokser  
> Prøv gratis på vinbot.dk

### Tekst til undertekster (ca. 7 sek. pr. slide)

1. **Find vin på sekunder** — Skriv ret, drue, stemning eller budget på vinbot.dk  
2. **Filtrér resultaterne** — Vælg stil, forhandler og pris — klik videre til butikken  
3. **Læs vinguider** — Madparring, druer og regioner — hundredvis af artikler  
4. **Opskrifter med vin** — Coq au vin, risotto og mere — med vin i gryden  
5. **Vinkøleskabe** — Søg modeller hos Vinkøleskabet.dk  
6. **Prøv Vinbot** — vinbot.dk

Skærmbilleder ligger i `video-source/`.
